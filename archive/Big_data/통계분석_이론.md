# 통계학 이론

## 통계학의 개요

데이터 과학(Data science) 을 위한 통계학이란?
<br/>
`데이터 과학 : 다양한 분야의 지식과 기술을 활용하여`
<br/>
`데이터를 통해 지식과 통찰력을 얻는 융합적인 분야.`

<br/>

- 실제 문제를 통한 동기 유발
- 데이터 시각화와 탐색적 자료 분석
- 데이터를 통해 알 수 있는 정보(내재되어 있는 편향 , 인과관계 , 상관관계 등)
- 모형과 알고리즘
- 확률이론을 바탕으로 한 예측 등
<br/>


## 사례 : 해롤드 시프먼

해롤드시프먼(Harold Shipman)은 영국역사상 가장 많은사람을 살해한 살인범으로<br/>
맨체스터 교외에서 온화한 가정의(family doctor)로 알려져있었음.<br/>
<br/>
1975년부터 1998년사이 적어도 215명을 살해한 것으로 확인되며<br/>
45명의 추가살인도 의심되었음.<br/>
<br/>
희생자 중 한명의 유언장을 위조한 것을 희생자의 딸(변호사)이 발견하여<br/>
경찰에 신고하여 덜미를 잡히게 됌.

## PPDAC 프레임 워크

PPDAC 는 통계 분석을 하기위한 기법 중 하나이다.<br/>
<br/>

Problem(문제정의) : 해결해야 할 문제가 무엇인지 정의하는 과정.<br/>
Plan(계획 수립) : 데이터 수집 및 분석을 어떠한 방법으로 할지 설계하는 것.<br/>
Data(데이터 수집) : 데이터 분석에 필요한 데이터를 수집하는 것.<br/>
Analysis(데이터 분석) : 수집한 데이터를 통계기법에 따라 분석하는 것.<br/>
Conclusion(결론 도출) : 분석한 내용을 바탕으로 최종적으로 해석, 의사결정.<br/>

PPDAC 를 풀어내면 이러한 이론을 알 수 있는데,<br/>
통계 분석을 하기위한 이론이다.

## 해롤드 시드먼의 사례 파악 및 통계 분석

이제 PPDAC 를 바탕으로 해롤드 시프먼의 사례를 풀어보겠다.<br/>
<br/>
Problem : 해롤드 시드먼은 얼마나 많은 사람을 살해하였는가?<br/>
Plan : 해롤드 시드먼의 환자 중 사망한 사람들의 특징 등을 파악한다.<br/>
Data : 환자의 자료를 파악해서 사망한 사람들의 특징 데이터를 정제한다.<br/>
Analysis : 데이터 시각화를 통한 탐색적 데이터 분석 가설검정을 수행함. <br/>
Conclusion : 해롤드 시프먼은 최소215명을 살해하였으며 45명이추가로<br/>
살해되었을 가능성이 높음.<br/>
<br/>

# 범주형 데이터란?

범주 데이터 : 2개 이상의 범주를 가지는 변수를 의미<br/>
<br/>
순서가 없는 데이터(숫자 데이터가 아닌 경우)<br/>
학과 , 이름 등<br/>
<br/>
순서가 있는 데이터(숫자 데이터가 아닌 경우)<br/>
이등병,일병,상병,병장 등<br/>
<br/>
순서가 있는 데이터(숫자데이터인 경우)<br/>
학년을 1 , 2 등으로 특정 범주로 나눈 후 순서대로 나열한 경우.<br/>
<br/>

## 상대 위험도

예를 들어 흡연을 했을 경우,<br/>
비흡연자에 비해 폐암 발병률이 1400% 더 높다.<br/>
여기서 1400% 가 상대 위험도이다.<br/>
<br/>

## 절대 위험도

상대 위험도에 대비되는 개념인데,<br/>
절대 위험도는 각 집단에서 위험에 처해지는 비율을 얘기한다.<br/>
<br/>
예를 들어 100명의 집단이 있을 때<br/>
1명이 비흡연자이면서 폐암에 걸린다 가정했을 때,<br/>
14명의 흡연자가 폐암에 걸린다고 하면,<br/>
절대 위험도는 100명중 15명이므로 15% 인 것이다.<br/>
<br/>
<br/>

# 수치형 데이터란?

범주형 데이터는 이름이 붙는 데이터이고,<br/>
수치형 데이터는 숫자로 이루어지며 의미가 있는 데이터를 일컫는다.<br/>
<br/>

## 이산형 데이터와 연속형 데이터

이산형 : 값이 정수인 경우(횟수 , 개수 등)<br/>
연속형 : 값이 실수인 경우(길이 , 무게 등)<br/>
<br/>

## 시각화

데이터가 많은 경우 일일히 데이터를 전부 뒤져보며 연구할 수 없다.<br/>
이럴 경우 자료의 특징을 나타내는 몇가지 '요약 값'을 제시할 수 있는데,<br/>
이러한 값을 통계량 이라고 한다.<br/>
<br/>
또한 데이터가 가지는 모든 값의 형태는 분포라고 한다.<br/>
통계량을 한눈에 보기쉽게 그림 등으로 나타내는 것을 시각화라고 한다.<br/>
<br/>

## 시각화의 종류

![시각화종류](https://github.com/user-attachments/assets/117c2464-aaf2-4ed8-bd2d-8b9436b03db9)

a 처럼 점으로 나타낸 그래프를 strip 차트라고 한다,<br/>
값의 분포를 정확하게 알 수 있지만<br/>
많은 데이터가 분포되어 있을 경우<br/>
한 곳에 많은 데이터가 뭉쳐져 보기 힘들 수 있다.
<br/>

b 처럼 박스를 중앙에 그려 데이터의 분포를 나타낸 차트를<br/>
상자그림(box plot)이라고 한다.<br/>
박스를 중앙에 두고,<br/>
한쪽엔 최댓 값 , 반대쪽엔 최솟 값을 둔채로<br/>
중앙 부분에 중앙 값 부분에 네모를 그려 표현한다.<br/>
<br/>

c 처럼 막대그래프 형태로 데이터의 분포를 나타낸 차트를<br/>
히스토그램이라고 한다.<br/>
하지만 일반 막대그래프와는 조금 다른 개념이다.<br/>
<br/>
막대그래프는 세로의 길이만을 생각하지만<br/>
히스토그램은 가로 , 세로에 범주를 나눠 시각화를 하기 때문에<br/>
가로와 세로의 높이를 전부 고려해서 시각화를 해야한다.<br/>
<br/>

# 좋은 데이터 시각화의 특징.

- 믿을만한 정보를 담고있다.

- 유의미한 패턴이 뚜렷하게 나타남.

- 관심을 끌 수 있도록 눈에 띄는 것이 중요 하지만 데이터가 나타내는 의미가 훼손되서는 안됨.

- 필요시 추가 탐색이 가능하도록 만들어야 함.

## 성공적인 데이터 시각화의 방법

### 기능성과 심미성의 균형

`심미 : 아름다움을 살펴 찾음`<br/>

기능과 심미의 균형을 잃을 경우 목적하는 바를 이루지 못할 수 있음.

### 사용자의 정보 인지방법 이해

사용자가 어떠한 정보를 인지하는지 파악한다.<br/>
사용자는 어떠한 기준으로 볼 지 파악한다.

인간이 시각적 형태를 지각하는 방법을 설명한 것.<br/>
인간의 시각적 인식이 어떻게 작용하는지 구체적인 근거를 제시함.

### 군집의 원리 종류

대표적인 5가지 원리는 파악해보자.<br/>
`근접` `유사` `폐쇄` `연속` `공동운명` 등이 있다.<br/>

# 스프레드시트

많은 사람들이 자료의 입력 및 분석을 위해 '스프레드 시트'를 사용한다.<br/>
<br/>

스프레드 시트 작성 시 유의할점을 알아보자.<br/>
<br/>

파일명은 'a1_2025-04-22' , 'a2_2025-04-30' 등으로 일관성있게 작성한다.<br/>
날짜 표기의 방식을 동일하게 한다. (YYYY-MM-DD) 로 하는 것이 일반적.<br/>
<br/>
주석 표기를 명확하게 한다.<br/>
`"Hello" , "hello"`를 혼용하면 헷갈리므로 이러한 작성은 지양한다.<br/>
<br/>
셀안의 여분의 공간 `"hello" , " hello"` 을 조심한다.<br/>
<br/>
변수나 파일의 이름에 특수문자 ` -  , _ ` 를 빼고 쓰지 않는다.<br/>

### 백업파일을 만들어라.

<br/>

# 귀납적 추론

`귀납 : 개별적인 사실,경험등을 바탕으로 이론적인 추론을 유도 함.`<br/>
<br/>
이러한 귀납으로 추론을 할 수 있는데<br/>
귀납적 추론에는 4가지 법칙이 있다.<br/>
<br/>
4가지 법칙의 초성은 DSST이다.<br/>
<br/>
전국 고등학생들의 스트레스에 관해 설문할 때,<br/>
데이터(Data) : 학생들이 제출한 설문 내용<br/>
표본(Sample) : 10개 고등학교에서 각 학교에서 랜덤으로 100명씩, 총 1000명의 학생들<br/>
연구모집단(Study Population):서울 , 경기소재 총 10개 학교의 고등학생들<br/>
목표 모집단(Target Population):전체 고등학생들(일반화 되게 될 표본)<br/>
<br/>

### 내적 타당성
(표본 -> 연구모집단) 중 발생함<br/>
표본이 연구모집단의 대표성을 가질 경우<br/>
내적 타당성을 지닌다고 할 수 있다.<br/>
<br/>

### 외적 타당성
(연구모집단 -> 목표 모집단) 중 발생함<br/>
연구 모집단이 목표 모집단의 특성을 가져<br/>
목표모집단으로 확장할 수 있는 경우 외적 타당성을 지닌다고 한다.

## 센서스

표본을 선택하지만 전체모집단을 조사하는 경우를 센서스라고 한다.<br/>
<br/>
Ex) 교내 전체 학생들의 건강조사<br/>
<br/>
하지만 센서스는 문제점이 몇 가지 있다.<br/>
<br/>

- 센서스는 전체 모집단을 조사하므로 조사에 잡히지 않는 특정 집단이 존재할 수 있음.
- 조사 중 모집단에 변수(출생 , 사망)등이 발생할 수 있으므로 완벽히 모집단의<br/>
모든 사람을 조사하는 것은 불가능함.
- 따라서 센서스보다 표본조사 방법이 복잡성 , 비용 측면에서 더 유리함.

## 표본 편의
### 무응답 편의

전체 모집단중 임의로 뽑힌 사람들 중 일부만 대답하는 경우<br/>
전체 모집단을 대표할 수 없다.

### 자원응답 편의

관심이 있는 사람들만 조사에 응답하거나,<br/>
선물을 받기위해 응답하는 경우

### 선택 편의

표본을 잘못 선택하여 연구모집단과 목표모집단이 상이하는 경우 발생한다.

## 전통적인 임의 표본추출 방법

크게 4가지가 있는데,<br/>
단 층 집 다 로 외우자.<br/>

### 단순임의추출

모집단을 n개의 표본로 나눌 때<br/>
각 표본의 비율이 모두 일정하도록 랜덤으로 추출하는 기법.

### 층화추출

비슷한 관측치가 나타나는 구역을 각 층으로 나누어<br/>
공정하게 추출한다.<br/>
Ex) 전교생이 1000명인 학교에서 100명을 뽑아야 할 때,<br/>
남자 600 명 , 여자 400 명이면,<br/>
남자 (600/1000) = 60명 , 여자 (400/1000) = 40명 뽑을 수 있음.

### 집락추출

모집단을 집락으로 나눈 다음,<br/>
그 집락 안에서 모두 혹은 일부를 조사하는 기법.<br/>
Ex)전체 초등학생을 조사한다고 했을 때<br/>
전부를 무작위로 뽑기엔 좀 그러니, 각 학교를 '집락'으로 생각하여<br/>
각 학교내에서 모든 학생이나 일부 학생을 조사한다.

### 다단계 추출

위에 나온 표본추출 기법을 효율적으로 섞어쓴다.<br/>
차이점은 한번에 추출하려는 방식이 아닌,<br/>
절차에 따라 단계별로 하나하나 걸러낸다.<br/>

## 기타 표본추출방법

### 포획 - 재포획 추출

조금 불쌍한 개념이긴 하지만,<br/>
n개의 동물을 잡고 각 동물마나 표지를 부착한다.<br/>
포획한 동물을 야생에 풀어주고 다시 K개의 동물을 잡아들인다.<br/>
여기서 표지가 부착되어있는 동물은 k라고 한다.

### 트란섹트 추출

선을 한번 긋고, 그 선을 따라 가면서<br/>
일정 간격마다 야생동물의 분포 등을 조사한다.

## 정규 분포

이젠 추출 기법을 떠나 분포의 종류를 알아보자.<br/>
정규 분포는 종 모양의 그래프로 표현되는데<br/>
그래서 '벨 커브(bell curve)'라고 불리기도 한다.<br/>
<br/>
값들이 평균을 중심으로 균등하게 분포되어있는 형태로 되어있다.<br/>
평균을 중심으로 데이터가 고르게 분포되어있어서<br/>
평균 , 중앙 값 , 최빈 값이 모두 같다.<br/>
<br/>
당연히 평균과 어느정도 다른 값이 존재하므로,<br/>
표준편차라는 것이 존재하는데, 총 3가지로 나눈다.<br/>
<br/>
표준편차의 값에 따라 법칙이 하나 존재한다.<br/>
68-95-99.7 법칙이 있다.<br/>
전체 데이터의 68%는 표준편차 (+1,-1) 내에 존재,<br/>
전체 데이터의 95%는 표준편차 (+2,-2) 내에 존재,<br/>
전체 데이터의 99.7%는 표준편차 (+3,-3) 내에 존재,<br/>
<br/>
모집단에서 모평균 , 모표준편차를 '모수'라고 부르며<br/>
2가지 모수에 의해 그래프의 모양이 결정된다.
