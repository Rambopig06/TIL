- 알고리즘의 성능을 측정하는 방법으로 '시간 복잡도(Time Complexity)' 가 있다.

```
합계 <- 0

for 숫자가 1부터 100까지 반복 :
  합계 <- 합계+숫자

합계 출력
```
**(알고리즘 1)**
```
합계 <- 0
합계 <- (1+100) * (100) /2

```

## 4. 자료구조

- 자료구조란, 컴퓨터 프로그래밍에서 효율적인 자료(데이터)의 형태를 말한다.

#### 자료구조의 형태

<img width="693" height="214" alt="image" src="https://github.com/user-attachments/assets/650436a2-cf6c-4ce0-ae68-1a061163f7ec" />

- 선형 자료 구조 : 데이터를 한줄로 순차적으로 표현한 형태
- 비선형 자료 구조 : 하나의 데이터 뒤에 여러개가 이어진 형태(Ex.마인드맵)
- 파일 자료 구조(순차파일) : 파일 내용을 논리적인 처리순서에 따라 연속해서 저장
- 파일 자료 구조(직접파일) : 해시 함수로 내용 저장 위치를 계산하여 저장
- 파일 자료 구조(색인 순차 파일) : 순차 파일과 직접 파일이 결합된 형태
