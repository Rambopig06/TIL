# 파이썬 함수

이번 주는 파이썬을 중점으로 작성할 예정이다.<br/>
필자가 파이썬을 집중적으로 파야하는 주간이기 때문..<br/>
<br/>
영어를 공부하고 영어로 영어권 사람들과 말을 섞기 위해선,<br/>
회화법을 알아야하고, 회화법을 공부하기 위해<br/>
제일 기초가 되는 것은 바로 영단어를 암기하는 것이다.<br/>
<br/>
문장을 만들기 위해선 단어의 뜻, 문법 등등을 알아야 하기 때문.<br/>
<br/>
파이썬에서 코드를 작성시키고 컴퓨터와 상호작용 하기 위해서도<br/>
영어를 할 때와 똑같이 함수 , 문법 등등을 알아야 한다.<br/>
<br/>
오늘은 프로그래밍의 기초라고도 할 수 있는<br/>
'함수'를 알아볼 것이다.<br/>
<br/>


| 함수 이름         | 하는 일       | 쉬운 설명 예시                                        |
| ------------- | ---------- | ----------------------------------------------- |
| `print()`     | 화면에 보여줌    | `print("ㅎㅇ")` → ㅎㅇ                              |
| `input()`     | 사용자 입력 받기  | `name = input()` → 이름을 직접 입력                    |
| `len()`       | 길이 세기      | `len("apple")` → 5                              |
| `sum()`       | 숫자 더하기     | `sum([1, 2, 3])` → 6                            |
| `max()`       | 가장 큰 값     | `max(3, 7, 1)` → 7                              |
| `min()`       | 가장 작은 값    | `min(3, 7, 1)` → 1                              |
| `sorted()`    | 정렬하기       | `sorted([3, 1, 2])` → \[1, 2, 3]                |
| `str()`       | 문자로 바꾸기    | `str(123)` → "123"                              |
| `int()`       | 정수로 바꾸기    | `int("123")` → 123                              |
| `float()`     | 소수로 바꾸기    | `float("3.14")` → 3.14                          |
| `set()`       | 중복 제거 (집합) | `set("aabb")` → {'a', 'b'}                      |
| `type()`      | 자료형 알려줌    | `type(3)` → `<class 'int'>`                     |
| `range()`     | 숫자 순서 만들기  | `range(5)` → 0부터 4까지                            |
| `reversed()`  | 거꾸로 뒤집기    | `reversed([1,2,3])` → \[3,2,1]                  |

<br/>

### print()
제일 먼저 print 함수는 SNS 코딩 밈에서도 제일 많이 떠도는<br/>
국룰같은 명령어이다. <br/>
```
print("hello world")

-> hello world
```

<br/>
요 명령어 하나만큼은 비전공자도 안다.<br/>
print를 작성하고 괄호안에 출력할 내용을 입력하면 된다.<br/>
<br/>
<br/>
<br/>
<br/>

### input(사용자 입력 받기)

input 명령어는 사용자에게 값을 입력받는다.<br/>
표에 나온 것처럼 이렇게 쓰면,<br/>
```
name = input("이름을 입력하세요:")

-> 이름을 입력하세요:
```
이렇게 코드를 작성하면,<br/>
프롬포트에 `이름을 입력하세요:` 라고 출력되고,<br/>
사용자는 프롬포트에 값을 입력하는 상태가 된다.<br/>
<br/>
그리고 입력한 값은 name 이라는 '변수'에 저장된다.<br/>
저장된 값은 name 이라는 변수를 잘 활용해서 사용하면 된다.<br/>
<br/>
<br/>
<br/>
<br/>

### len(길이 세기)

len 함수는 길이를 의미하는 length 에서 나온 함수이다.<br/>
<br/>
```
print(len("안녕하세요"))

-> 5
```
이렇게 입력하면 프롬포트에는 5가 출력된다.<br/>
시험 준비하면서 글을 쓰는지라 사진까지 올릴 여유는 없지만,<br/>
필자가 직접 일일히 실행하며 쓰는 글이니<br/>
믿고 보면 된다.<br/>
<br/>

```
print(len("안녕 하세요 "))

-> 7
```

이렇게 띄어쓰기를 한채 실행 시키면<br/>
7이 출력된다.<br/>
<br/>
len 함수는 띄어쓰기 및 공백을 카운팅하기 때문.<br/>
<br/>
<br/>
<br/>
<br/>

### sum(숫자 더하기)

```
print(sum([1,2,3]))

-> 6
```

1 + 2 + 3 의 값을 구하고 출력하는 코드이다.<br/>
위에 서술한 방식은 1 , 2 , 3 과 같은 요소들을<br/>
"리스트" 안에 담은 방식이고,<br/>
<br/>

```
print(sum((1,2,3)))

-> 6
```
이런식으로 소괄호에 담아 전달하는 방식은<br/>
"튜플" 이라고 한다.<br/>
<br/>
만약 대괄호로 묶어 리스트화 시키거나<br/>
소괄호로 묶어 튜플화를 시키지 않는다면<br/>
에러가 난다.<br/>

```
print(sum(1,2,3))

-> Error
```
<br/>
왜냐하면 `sum(1,2,3)` 이런식으로,<br/>
sum 이라는 함수는 바로 숫자만 띡 주면 받는 그런 함수가 아니다.<br/>
<br/>
쉽게 비유하자면 sum 함수는 입이 고급져서<br/>
단순 숫자형은 먹지않고 리스트 , 튜플 자료형만 취급한다.<br/>
<br/>
<br/>
<br/>
<br/>

### min , max(최솟값 , 최댓값)

위에 서술한 sum 함수는<br/>
튜플 , 리스트 자료형만 취급했지만,<br/>
이 친구들은 함수 뒤 괄호에 바로 숫자형을 써도 된다.<br/>
<br/>
물론 튜플 , 리스트로 적어도 정상 작동한다.<br/>
<br/>

#### min(최솟값 출력 함수)

```
print(min(1,3,5,7))

-> 1
```

<br/>
<br/>

#### max(최댓값 출력 함수)

```
print(max(1,3,5,7))

-> 7
```

<br/>
<br/>
<br/>
<br/>


### sorted(정렬 함수)

다시 취향 까다로운 녀석으로 돌아왔다.<br/>
sum 과 같이 리스트 , 튜플 자료형만 받는 아이다.<br/>
<br/>

```
print(sorted((4,7,3,9,1)))

-> [1,3,4,7,9]
```

튜플이 아니라 리스트형으로 작성해도 결과는 같다.<br/>
이처럼 숫자의 대소유무를 파악하여<br/>
작은 순부터 큰 순까지 정렬하는 함수이다.<br/>
<br/>
<br/>
<br/>
<br/>

### str(문자형으로 변환) , int(숫자형으로 변환)

이 글을 읽는 독자라면<br/>
이젠 자료형에 대해서도 어느정도 알고 있을 것이므로,<br/>
바로 숫자형 , 문자형 자료형태 변환에 대한 함수를 공부 해보자.<br/>
<br/>
<br/>

#### str(문자형으로 변환)
```
print(str(123))

-> 123
```
이 코드는 123 이라는 '숫자'를<br/>
문자 '123' 으로 출력한 것이다.<br/>
<br/>
육안으로 확인하기 힘들지만,<br/>
테스트를 해볼 수 있다.<br/>
<br/>
```
age = str(123)

print(13 + age)

-> Error
```

123 이라는 숫자를 str 함수안에 묶고<br/>
age 라는 변수에 저장한 뒤,<br/>
print로 13과 age 값을 더해본다.<br/>
<br/>
age 의 값은 12이고,<br/>
13 + 12 이므로 25가 출력될줄 알았지만 에러를 표출한다.<br/>
<br/>
서로의 자료형이 다르기 때문.<br/>
<br/>
<br/>

#### int(숫자형으로 변환)

```
age = int("12")

print(13 + age)

-> 25
```

int 함수는 문자형을 숫자형으로 변환하는 함수이다.<br/>
\"" 를 이용하여 12를 문자형으로 만들었지만,<br/>
int함수가 그런 문자형 12를 숫자형으로 바꾸어준 것이다.<br/>
<br/>
따라서 print 에서 13 + age 는 13 + 12로 계산되므로,<br/>
25가 출력된다.<br/>
<br/>
하지만 의문이 들 것이다.<br/>
<br/>
"int 함수는 모든 문자형을 숫자형으로 바꿀 수 있나..?"<br/>
<br/>
당연히 아니다.<br/>


```
age = int("하하")

print(13+age)

-> invalid literal for int() with base 10: '하하'
```
에러 메시지에 표출된 'invalid literal for int() with base 10: '하하'<br/>
이 의미는 int 로 10진수로 만들려고 했던 요소가 '하하' 였지만,<br/>
'하하' 는 10진수로 바꿀 수 없는 형태이다.<br/>
<br/>
같은 문자형이라고 전부 바꿔지는 것이 아닌,<br/>
숫자처럼 생긴 문자형만 변환되는 것이다.<br/>
<br/>
필자 특유의 이상한 비유로 따지자면,<br/>
<br/>
아이돌처럼 얼굴이 미인인 일반인은,<br/>
아이돌 소속사에 들어가 아이돌이 될 수 있지만,<br/>
그렇지 않은 일반인은 아이돌이 되기 힘든 것처럼<br/>
같은 일반인이라고 같은 일반인이 아니라는 뜻이다.<br/>
<br/>
아이돌처럼 생겨야 아이돌이 될 수 있다는...<br/>
이상한 비유를 한번 끄적여봤다.<br/>
<br/>
<br/>
<br/>
<br/>

### float(소수형으로 변환)

지금까지 정수형 숫자는 다뤘지만<br/>
소수형 숫자를 다루지는 않았었다.<br/>
<br/>
자료형중에는 소수를 나타내는 float도 있으므로<br/>
한번 배워보자.

```
print(float("3.14"))

-> 3.14
```

문자형인 3.14 를 소수형으로 바꾸었다.<br/>
이번엔 큰따옴표 없이 3.14만 입력 후,<br/>
Type 함수를 이용하여 자료형을 확인해보면<br/>
<br/>

```
print(type(3.14))

-> float
```

float(소수형) 이 출력된다.<br/>
<br/>
<br/>
<br/>
<br/>

### range (숫자 순서 만들기)

```
a = range(4)

print(list(a))
```

이런식으로 사용하면 되는데, range 가<br/>
숫자를 일정한 규칙에 따라 순서대로 리스트에 정렬하는 함수이므로,<br/>
리스트를 붙여서 명령하여야한다.<br/>
<br/>

range 뒤 괄호에 넣는 숫자도 의미가 있다.<br/>
`(8)` 일정부분부터 괄호에 넣는 끝 숫자보다 1만큼 덜 출력된다.<br/>

```
a = range(8)
print(list(a))

-> [0,1,2,3,4,5,6,7]
```
이런식으로 시작 값을 지정하지 않으면<br/>
0부터 차례대로 리스트에 정렬이 되는데,<br/>
끝 값은 괄호에 적은 8보다 1 작은 값인 7까지 출력된다.<br/>
<br/>
<br/>

`(2 , 8)`<br/>
이건 2부터 끝 값인 (8-1)까지 출력한다는 뜻이다.<br/>
괄호안에 2개의 값이 있을 때,<br/>
(시작 값 , 끝 값) 순이다.<br/>
<br/>

`(1,9,2)`

이건 1부터 (9-1)까지 2의 간격으로 차례대로 출력한다.<br/>

```
a = range(1,9,2)
print(list(a))

-> [1,3,5,7]
```

<br/>
<br/>
<br/>

### reverse , reversed(리스트 뒤집기)

```
a = [1,4,7,9]
print(reversed(a))
```

이 방법대로 하면 정상실행되지 않는다.<br/>
<br/>

```
a = [1,4,7,9]
a.reversed()

print(list(a))

-> [9,7,4,1]
```
